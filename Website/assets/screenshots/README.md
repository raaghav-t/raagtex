# App Screenshots

Current screenshot filenames used by the website:

- `justLaunched.png`
- `editorLightMode.png`
- `editorDarkMode.png`
- `stackedView.png`

If you rename files, update the `src` paths in `Website/index.html`.

Recommended capture size:
- Width: 2200-2800 px
- Aspect ratio: 16:10 or 16:9
- Format: PNG for sharp UI text
